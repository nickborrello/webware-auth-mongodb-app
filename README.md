# Express MongoDB Auth App

A full-stack Node.js application featuring persistent storage and user authentication.

## Features
- **Authentication**: User login using Passport.js (Local + Google OAuth).
- **Persistence**: MongoDB database integration for storing user data.
- **Templating**: Server-side rendering with EJS.
- **Security**: Password hashing with bcrypt.

## Tech Stack
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Auth**: Passport, bcrypt
- **Views**: EJS

## Setup
1. Install dependencies: `npm install`
2. Configure `.env` with DB credentials and API keys.
3. Run: `npm start`
